# InsectMotion
A user-friendly Python-based tool that helps analyze muscle dynamics and other movements from the video files.
Originally developed to analyze movements in Insects, however it can be applied to detect any movement in the video.
This app offers 2 analyses:
  one produce data (numbers) that reflect the intensity of movements in the selected ROIs along with peak analysis;
  another produce png image that visualize active (white) vs. inactive (black) pixels along with ROI metrics
Both analyses are customizable, allowing users to optimize the settings to best fit the requirements of their experiment.
